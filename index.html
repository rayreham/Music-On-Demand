☁️MusicOnDemand☁️

Use the search bar to access the Apple Music Api and find your favorite music!

~Inspo~

This web application allows you to search the Apple music API for songs, artists, a nd albums. I am an avid listener of music of all genres and have come across many music platforms that do not have artists that I listen to. I decided to use this project to develop a platform that will allow me to preview artists' songs without downloading their application blindly. This was an incredibly fun and informative project to work on :)
